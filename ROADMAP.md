
# 后续开发计划（迭代路线）

以下功能为后续迭代规划，会在稳定版本逐步实现：

## 1. 安全性增强（计划高优先级）

| 功能 | 说明 |
|---|---|
| Token 鉴权 | 在访问上传/编辑接口时需要 Token |
| 临时访问链接 | 可生成一次性/限时 URL 用于分享 |
| HTTPS 支持 | 增加反向代理 + TLS 证书支持 |

---

## 2. 配置管理增强（计划中）

| 功能 | 说明 |
|---|---|
| 自动备份历史版本 | 文件覆盖前自动备份并可回滚 |
| 文件差异查看 | 对比新旧配置内容差异 |
| 批量上传/下载 | 整体同步多个 Clash 配置 |

---

## 3. 用户体系与多设备支持（可选）

| 功能 | 说明 |
|---|---|
| 用户登录 | 支持本地用户表 |
| 多用户隔离仓库 | 不同用户不同配置空间 |

---

## 4. UI 体验优化

| 功能 | 说明 |
|---|---|
| 夜间模式 | 自适应深色主题 |
| 文件重命名支持 | 改名不改变导入链接（软链接方式） |
| 自定义二维码样式 | logo / 标签注释 |

---

## 当前版本状态

- 当前版本用于个人本地或内网环境，稳定可用
- 尚未加入安全认证体系，请勿裸露公网部署
